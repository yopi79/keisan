# 制作スケジュール逆算計算機

リリース日を入力すると、営業日ベース（土日＋日本の祝日除外）で逆算して制作スケジュールを自動計算するブラウザツールです。

## 使い方

`index.html` をブラウザで開くだけ。GitHub Pages を有効化すれば以下のURLで公開できます。

```
https://yopi79.github.io/keisan/
```

### GitHub Pages 有効化手順
1. リポジトリの **Settings** → **Pages**
2. **Source** を `Deploy from a branch` に
3. **Branch** を `main` / `/ (root)` に設定して **Save**

## 機能

- リリース日から逆算してスケジュールを自動計算
- 土日＋日本の祝日（2025〜2027年分内蔵）を営業日から除外
- デフォルト6工程（要件定義 → デザイン → コーディング → 社内確認 → 修正 → 最終確認・公開準備）
- 工程名・日数のカスタマイズ、工程の追加・削除
- リリース日が休日なら直前営業日に自動調整して警告
- カスタム休日（夏季休暇など）を追加可能
- CSVダウンロード／テキストコピー／印刷（PDF保存）対応
